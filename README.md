# Tarea-Concurrencia-SSSM
Simulación de concurrencia en módulos del sistema SSSM de Medellín utilizando Python. El programa implementa hilos (threads) y semáforos para controlar el acceso concurrente a recursos compartidos y evitar condiciones de carrera.
